# Example using @unmock service repository

Petstore service was installed from [npm](https://www.npmjs.com/package/@unmock/petstore) with

```bash
npm i @unmock/petstore -D
yarn add @unmock/petstore -D
```

## Instructions

Install dependencies:

```
npm i
yarn // If using yarn
```

Run tests:

```
npm run test
yarn test
```
